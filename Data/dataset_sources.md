# Suggested datasets

The following public datasets can be used to test the Dynamical Subject Metric (Σ).

## Sleep EEG datasets

Sleep-EDF Database  
https://physionet.org/content/sleep-edfx/

## Anesthesia datasets

Propofol anesthesia EEG recordings  
https://physionet.org/

## Resting-state EEG

OpenNeuro datasets  
https://openneuro.org/

These datasets allow testing the prediction:

Σ_awake > Σ_sleep > Σ_anesthesia
